Preguntas:

1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

4. ¿Cómo se declara una variable con alcance local dentro de una función?

Respuesta:

1. Java es un lenguaje enfocado a la programación orientado a objetos y está fuertemente tipado, mientras que Javascript es un lenguaje basado en prototipos y es débilmente tipado.

2. getDate(), getDay(), getFullYear(), getMilliseconds() y getTime().

3. map(), filter(), find(), join() y push().

4. Hay dos formas de que una variable solo tenga alcance local en Javascript. La primera es declarar la variable con var dentro de una función y la segunda es declarar la variable con let.
