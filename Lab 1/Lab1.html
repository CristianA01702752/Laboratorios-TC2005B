Sobre el ciclo de vida y desarrollo de los sistemas de información:
14. ¿Cuál es el ciclo de vida de los sistemas de información?
15. ¿Cuál es el ciclo de desarrollo de sistemas de información?

Respuestas:

1. El internet es un sistema o red de dispositivos que están interconectadas. Mientras que la World Wide Web es una colección de páginas que funciona dentro del internet.
2. Un URL tiene 4 partes: Protocolo de comunicación (http o https), dominio, ruta de acceso y parámetros
3. Get: sirve para recuperar un dato específico, Head: sirve para eviar solo el header, Post: sirve para enviar información, Patch: sirve para actualizar parcialmente un recurso, Put: sirve para modificar información y Delete: sirve para eliminar recursos.
4. Se utiliza el método Post ya que este método envía los datos por el protocolo de comunicación, así se puede utilizar el protocolo HTTPS para mantener seguro los datos.
5. Se utiliza el método Get ya que este sirve para obtener datos específicos de un servidor.
6. Un código 200 significa que la solicitud o el método que se esté utilizando fue exitoso.
7. En la gran mayoría de veces, la respuesta sería no ya que los errores (400-499) son de que el usuario está cometiendo el error pero hay excepciones donde el desarrollador debe corregirlo.
8. Si, porque los errores (500-599) es que el servidor está teneindo problemas para ejecutarse correctamente y el desarrollador debe corregirlo.
9. Son atributos que se consideran obsoletos y no se recomienda su uso. Ejemplos de atributos de HTML 4 que son desaprobados en HTML5 son: isindex, noframes y applet.
10. html 5 tiene soporte para audio y video, por lo que ya no es necesario utilizar tecnologías como flash. La sintaxis de html 5 es más simple que en html 4.
11. filas (tr), encabezados de columnas (th), contenido de tablas (td), extender columnas (colspan), extender filas (rowspan), leyendas de tablas (caption), título (thead), cuerpo de tabla (tbody) y pie de una tabla (tfoot).
12. Campos de entrada de texto, casillas de verificación, botones y selector de archivos.
13. Mi navegador (Google Chrome) tiene buen soporte para HTML 5 ya que obtuve 476/555 puntos.
14.El ciclo de vida consiste de 5 fases: nacimiento, desarrollo, operación, mantenimiento y muerte.
15. El ciclo de desarrollo consiste de 7 fases: planificación de concepto, definición de requisitos, diseño, fase de pruebas, implementación, mantenimiento y disposición.
